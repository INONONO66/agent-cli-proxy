# Contributing to Agent CLI Proxy

Thank you for your interest in contributing to Agent CLI Proxy!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/agent-cli-proxy.git`
3. Install dependencies: `bun install`
4. Create a feature branch: `git checkout -b feature/your-feature`

## Development

- Use `bun run build` to build the project
- Use `bun test` to run tests
- Use `bun run src/cli.ts` to test CLI commands locally

## Submitting Changes

1. Ensure all tests pass: `bun test`
2. Commit your changes with clear messages
3. Push to your fork and submit a pull request
4. Provide a clear description of the changes and why they're needed

## Code Style

- Follow the existing code style in the repository
- Use TypeScript for all new code
- Add tests for new functionality

## Reporting Issues

Please use the GitHub issue tracker to report bugs or suggest features. Include:
- A clear description of the issue
- Steps to reproduce (for bugs)
- Expected vs. actual behavior
- Your environment (OS, Node/Bun version, etc.)
